# Domande Frequenti (FAQ)

## Qual è la versione minima di Python richiesta?

La versione minima di Python richiesta è **3.8**.

## Dove posso trovare l'API Documentation?

L'API documentation è disponibile all'indirizzo `http://localhost:8000/swagger`.

## Come posso modificare le impostazioni del progetto?

Per modificare le impostazioni, modifica il file `.env` nella cartella del back-end.
