# Utilizzo Front-end Progetto Libreria

## Accesso al Front-end

Per accedere all'interfaccia utente, esegui il seguente comando:

1. Navegare alla cartella del progetto:
   ```sh
   cd /Users/simo/PycharmProjects/libreria/libreria-frontend
   ```

2. Avviare l'applicazione:
   ```sh
   npm run start
   ```

3. Aprire il browser e navigare su `http://localhost:3000`.

## Funzionalità

Il front-end offre le seguenti funzionalità:

- Visualizzazione catalogo libri
- Ricerca dei libri
- Dettaglio del libro
- Gestione del carrello
