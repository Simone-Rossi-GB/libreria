# Utilizzo Back-end Progetto Libreria

## Accesso al Back-end

Per accedere all'interfaccia di amministrazione, esegui il seguente comando:

1. Navegare alla cartella del progetto:
   ```sh
   cd /Users/simo/PycharmProjects/libreria/libreria-backend
   ```

2. Avviare l'applicazione:
   ```sh
   python manage.py runserver
   ```

3. Aprire il browser e navigare su `http://localhost:8000/admin`.

## Funzionalità

Il back-end offre le seguenti funzionalità:

- Gestione degli utenti
- Gestione dei libri
- Importazione di dati del libro
- Export dati del libro
